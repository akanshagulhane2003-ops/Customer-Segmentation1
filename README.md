# 🎯 Customer Segmentation Project

Segment customers based on behavior and demographics using K-Means clustering.

## 📌 Project Overview

This project uses unsupervised machine learning to group customers into distinct segments, enabling targeted marketing strategies and personalized customer experiences.

## 🔑 Key Features

- ✅ Clustering using Python (scikit-learn)
- ✅ Purchase pattern & preference analysis  
- ✅ Visual segment characterization
- ✅ Actionable business insights

## 🛠️ Tech Stack

- **Python 3.x**
- **Pandas** - Data manipulation
- **NumPy** - Numerical computing
- **Scikit-learn** - Machine learning
- **Matplotlib & Seaborn** - Data visualization
- **Jupyter Notebook** - Interactive development

## 📊 Dataset

- **Source:** Mall Customer Segmentation Data
- **Features:** Age, Gender, Annual Income, Spending Score, Purchase Frequency, Avg Order Value
- **Records:** 200+ customers
